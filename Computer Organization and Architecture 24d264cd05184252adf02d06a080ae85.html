Worked examples

  • Make a program in C++ that checks if the Sentence Is Pangram with no set/map.
    bool checkIfPangram(string sentence) {
        int i = 0;
        int set = 0;
        while(i<sentence.size() && set != 67108863) {
            int element = sentence[i]-97;
            set = set | (1 << element); // 0, 1, 11, 111, 111, 1111, ....  
            i++;
        }
        return set == 67108863; // 11111111111111111111111111
}
