The Code

"The Code" es un documental del 2001 que narra la historia del sistema operativo GNU/Linux, desde sus inicios como un proyecto personal de un estudiante de computación (1991) hasta su transformación en un sistema operativo de código abierto utilizado en todo el mundo (2001). El documental finlandés es narrado desde el punto de vista de las personas más importantes del movimiento del Software libre y «Open source».

Los cinco personajes más importantes que aparecen en el documental son:

  • Linus Torvalds
  • Richard Stallman
  • Robert Young
  • Eric Raymond
  • Theodore Ts'o

El personaje iniciador del movimiento del Software Libre es Richard Stallman, quien comenzó a trabajar en el proyecto GNU en 1983.

Linus Torvalds es el personaje que inició el desarrollo del núcleo del sistema operativo GNU/Linux en 1991.

La propiedad intelectual del sistema operativo GNU/Linux es compartida por una comunidad global de desarrolladores, usuarios y organizaciones sin fines de lucro que lo mantienen y mejoran. Esto es así porque la licencia es GPL v2.

La Linux Foundation se encarga de administrar y supervisar el desarrollo del núcleo del sistema operativo GNU/Linux, conocido como el kernel Linux. Sin embargo, el desarrollo del kernel es realizado alrededor del mundo por distintos individuos, empresas y gobiernos.

Las cuatro libertades del Software Libre son:

  • La libertad de ejecutar el programa para cualquier propósito
  • La libertad de estudiar cómo funciona el programa y modificarlo
  • La libertad de distribuir copias del programa
  • La libertad de mejorar el programa y publicar las mejoras para que otros se beneficien.

Ventajas y deventajas del software libre comparado con el software propetiario:

  1. Ventajas del Software Libre:
  • Libre acceso al código fuente: si no te agrada algo, cámbialo.
  • Flexibilidad para personalizar y adaptar el software a necesidades específicas
  • Comunidad global de desarrolladores y usuarios que colaboran para mejorar y actualizar el software
  • Menor costo en comparación con el software propietario.

Desventajas del Software Libre:

  • Puede requerir habilidades técnicas para instalar y utilizar el software.
  • La documentación y soporte técnico pueden ser limitados.
  • No todos los programas o aplicaciones están disponibles en versión de código abierto.

Ventajas del software propietario:

  • Fácil de usar e instalar.
  • Mayor cantidad de programas y aplicaciones disponibles.
  • Soporte técnico y documentación ampliamente disponible.

Desventajas del software propietario:

  • No se tiene acceso al código fuente.
  • Poca o nula flexibilidad para personalizar o adaptar el software.
  • Altos costos de licencias y actualizaciones.

Si Linux perteneciera a una empresa, muy probablemente perdería su carácter de código abierto y libre, lo que significaría que los usuarios no tendrían acceso al código fuente, limitando su capacidad de personalizar y adaptar el sistema operativo. Además, la empresa propietaria podría controlar las actualizaciones y el soporte técnico, aumentando los costos para los usuarios.
