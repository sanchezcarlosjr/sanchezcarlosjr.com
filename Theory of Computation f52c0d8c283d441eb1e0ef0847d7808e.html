Finite automata

Finite automata and their probabilistic counterpart Markov chains. Finite Automaton (FA) or Finite-State Machine. An automaton (Automata in plural).

  • The computer is complex, instead, we use an idealized computer called a computational model.
  • A finite automaton:

We feed the input string 010101﻿ to the machine M1M_1M1​﻿, the processing proceeds as follows:

  1. Start in state q1q_1q1​﻿.
  1. Read 1, follow the transition from q1q_1q1​﻿ to q2q_2q2​﻿.
  1. Read 1, follow the transition from q2q_2q2​﻿ to q1q_1q1​﻿.
  1. Accept because M1M_1M1​﻿ is in an accept state q2q_2q2​﻿ at the end of the input.

The formal definition of a deterministic finite automaton (DFA)

A finite automaton is a 5-tuple (Q,Σ,δ,q0,F)(Q,\Sigma,\delta,q_0,F)(Q,Σ,δ,q0​,F)﻿ where

  1. QQQ﻿ is a finite set called the states,
  1. Σ\SigmaΣ﻿ is a finite set called the alphabet,
  1. δ:Q×Σ→Q\delta:Q\times \Sigma\rarr Qδ:Q×Σ→Q﻿ is the transition function,
  1. q0∈Qq_0 \in Qq0​∈Q﻿ is the start state, and
  1. F⊆QF\subseteq QF⊆Q﻿ is the set accept states, they are sometimes called final states.

The formal definition of a nondeterministic finite automaton (NFA)

A finite automaton is a 5-tuple (Q,Σ,δ,q0,F)(Q,\Sigma,\delta,q_0,F)(Q,Σ,δ,q0​,F)﻿ where

  1. QQQ﻿ is a finite set called the states,
  1. Σ\SigmaΣ﻿ is a finite set called the alphabet,
  1. δ:Q×(Σ∪{λ})→P(Q)={R∣R⊆Q}\delta:Q\times (\Sigma\cup\{\lambda\})\rarr P(Q)=\{R|R \subseteq Q\}δ:Q×(Σ∪{λ})→P(Q)={R∣R⊆Q}﻿ is the transition function (P is a power set),
  1. q0∈Qq_0 \in Qq0​∈Q﻿ is the start state, and
  1. F⊆QF\subseteq QF⊆Q﻿ is the set accept states, they are sometimes called final states.
