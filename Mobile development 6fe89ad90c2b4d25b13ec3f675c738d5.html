How does it work?

Apache Cordova is a hybrid option and like other options, you can deploy it to iOS, Android, and Windows Phone with the same codebase and habitual native API device but no modern API and it’s slower than native options.

However, the most common use is as a foundation in other frameworks such as Ionic and native.
