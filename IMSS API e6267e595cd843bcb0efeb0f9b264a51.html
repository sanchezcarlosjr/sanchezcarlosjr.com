Overview

Obtain and verify worker from the IMSS by CURP or Mexican Name.

🕑 Average Latency ~100 ms 🕑 Worst Latency ~5000 ms 🧑‍⚖️ Success Rate 80%

Getting started

  1. Log in and copy API Key.
  1. You either can test directly in your browser, use Postman, or use our sandbox. Remember substitute API Key.

Request

GET https://us-west4-arsus-production.cloudfunctions.net/imss

Parameters

NameDescription
nssnumber 11 Security social number
apiKeystring You need to sign up on https://sanchezcarlosjr.com for you to get it.

Response

If successful, this method returns a response body with the following structure:

{
  "nss": "string",
  "idAssignmentNss": "number",
  "isRegistredIMSS": "boolean",
  "rightINC": "boolean",
  "rightSM": "boolean",
  "vigency": "date"
}

Properties

NameDescription
creditFromInfonavitnumber Monto de credito a otorgar. Moneda mexicana.
housingSubAccountBalancenumber Monto del Saldo de la Subcuenta de Vivienda. Moneda mexicana.
operatingExpensesnumber Monto de gastos de titulación, financieros y de operación. Moneda mexicana.
totalnumber Total aproximado de crédito. Moneda mexicana. total = creditFromInfonavit + housingSubAccountBalance -operatingExpenses
monthlySalaryDiscountnumber El descuento mensual de su salario para el fondo de protección de pago, la cuota de administración, el pago de ecotecnologías, el monto de seguro y seguro de danos. Moneda mexicana.
creditForEcotechnologiesnumber Monto máximo de crédito para ecotecnologías. Moneda mexicana.
productstring CI, CII Numero de credito
